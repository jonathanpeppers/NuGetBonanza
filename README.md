# .NET 10 NuGetBonanza

Put all the popular .NET MAUI packages in a sample and "root" them all!

Note that I mostly tested Android here, adding these popular packages to a .NET MAUI app:

```xml
<PackageReference Include="CommunityToolkit.Maui" Version="11.1.0" />
<PackageReference Include="CommunityToolkit.Mvvm" Version="8.4.0" />
<PackageReference Include="GoogleGson" Version="2.12.1" Condition=" '$(TargetPlatformIdentifier)' == 'Android' " />
<PackageReference Include="Newtonsoft.Json" Version="13.0.3" />
```

```bash
$ dotnet build -c Release -f net10.0-android -bl
```
